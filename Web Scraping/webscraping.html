What is Web Scraping?

Web scraping is the technique for extracting the relevant information from a webpage. web scraping allows user to get the relevant information from any websites and it can be done manually as well as with the help of bots as well.

    Why Python for Web Scraping?

    Python has become one of the widely used programming language and with its easy to learn syntax and huge number of libraries such as BeautifulSoup, Scrapy , Pandas etc, it is one of the most preferred programming language for web scraping.

    Things to know before getting started with Web Scraping

    • Web Scraping might be against the terms of service of some websites so it is suggested that first contact the administrator before starting web scraping.
    • Main step to start web scraping is to observe the webpage and inspect the source code of the website thoroughly.
    •  You should have some basic knowledge of HTML and CSS before starting web scraping because you will mostly deal with HTML and CSS code so this step is mandatory.

    2. Beautifulsoup

    BeautifulSoup is a python library used for parsing the HTML and XML data. We can iterated through all the data.
    You can read complete beautifulsoup documentation here

    For python 3:

    pip3 install requests

    For python 2.x

    pip install bs4

    Using BeautifulSoup Library

    import requests
    import bs4
    source = requests.get(url)
    soup = bs4.BeautifulSoup(source.text,lxml)

    select () : select() returns the list type object with the tag

    Command Description
    select('tag')
    		 tag is the name of any HTML tag
    select('.class_name')
    		 class name of the tag will always start with.
    select('#id')
    		 name of id of the tag will always start with #
    select('tag1 tag2')
    		 tag1 is the main tag while tag2 is the tag which is enclosed within the main tag
    select('tag1 > tag2')
    		 tag1 is the main tag while tag2 is the directly enclosed within the main tag
    • Using select() to grab the title of the webpage
    title = soup.select('title')
    • Using select() to grab the data from a specific tag
    tag = soup.select("div")
    • Using Select() to get the data from an attribute of a tag
    tag = soup.select("img")
    tag["src"]
    • Using select() to get data from the tag with specific id
    # id will always start with #
    id_data = soup.select("#id")
    • Using select() to get data from the tag with specific class
    # class will always start with .
    id_data = soup.select(".class")
